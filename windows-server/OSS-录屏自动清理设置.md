# 录屏 OSS 24 小时自动清理 — 阿里云控制台设置 (0 代码)

把"用户上传的录屏文件 24 小时后自动删除"这个事**交给阿里云 OSS 后台跑**, 你不用维护任何代码 / 不用写定时任务. 设置一次, 永久生效.

## 为什么要做

录屏 OSS 文件用途:
- 用户点 "转 mp4 下载" → 上传 webm → 后端转码 → 用户下载 mp4 → **下载完就没用了**
- 用户点 "进入剪辑" → 不上 OSS (纯内存, 你之前砍了这条 OSS 路径)

所以 OSS 里的录屏 webm/mp4 **都是临时文件**, 24h 后大概率没人来要了. 不删的话:
- 1000 用户 × 5 文件/天 × 200MB = 1 TB/月
- OSS 存储 ¥0.12/GB/月 = ¥120/月白花

设了 24h 自动删, OSS 后台扫前缀, 到期就清, 100% 可靠.

## 设置步骤 (5 分钟)

### Step 1: 进 OSS 控制台

[https://oss.console.aliyun.com](https://oss.console.aliyun.com) → 用你**个人账号** (因为 OSS 现在在个人账号下) 登录.

### Step 2: 找到 monoi 的 bucket

左侧 **Bucket 列表** → 点你存录屏的那个 bucket (一般叫 `monoi-xxx` 或类似).

### Step 3: 配生命周期规则

bucket 详情页, 左侧菜单:

```
基础设置
  └─ 生命周期         ← 点这个
```

页面右上角 **创建规则** 按钮 → 点.

### Step 4: 填规则参数

弹出表单, 按下面填:

| 字段 | 填什么 |
|---|---|
| **规则名称** | `录屏 24h 自动删` (随便取, 自己识别用) |
| **规则状态** | ☑ **启用** |
| **应用范围** | 选 **指定前缀** |
| **前缀** | `recordings/` (注意末尾的斜杠, 漏了会匹配错) |
| **基于最后修改时间** | ☑ 选这个 (不是基于创建时间) |
| **文件生命周期** | ☑ 删除文件 → 距最后修改 **1** 天 |
| **删除片段** (碎片) | ☑ 距最后修改 1 天 (顺便清未完成的分片上传) |

注意 **OSS 控制台的"文件生命周期"最小单位是 1 天**, 没有"24 小时"或"几小时". 1 天 = 24 小时, 等价.

### Step 5: 保存 → 立刻生效

点 **确定**, 规则就跑起来了. OSS 后台每天扫一次, 24 小时前的 `recordings/` 前缀对象就被清掉.

⚠️ **删除是不可逆的**. 用户必须**当天用完转 mp4 + 下载**, 不能"我下周再回来下".

但这本来就是录屏的预期 — 不是长期资料库, 是"录完即用即弃".

## 验证规则生效

设了规则之后, 你可以:

1. 上传一个测试文件到 `recordings/test.webm`
2. 控制台改这个文件的"最后修改时间" (高级文件管理可以改) 设到 25 小时前
3. 几分钟后 (OSS 扫不会立刻跑) 刷新, 文件应该消失

或者**简单点**: 设了规则就别管, 第二天看 `recordings/` 前缀下有没有过期的老文件没清.

## 配套: 后端 mp4 转码完, 删 webm 源文件 (可选优化)

当前流程:
```
用户录 webm → 上传 OSS (recordings/xxx.webm)
        → ffmpeg 转 mp4 → 上传 OSS (recordings/xxx_mp4.mp4)
        → 用户下载 mp4
```

webm 源文件用完就没用了, 但 24h 后才被生命周期清. 如果想**立刻删**节省存储, 可以改 main.py 的 `transcode_recording_to_mp4` 端点:
- 转码成功 + 上传 mp4 之后, 主动 `bucket.delete_object(src_key)` 删 webm
- mp4 留下来等 24h 生命周期清

但这个优化**省得不多** (一个文件多存 24h ≈ 0.1 GB × ¥0.12/GB/月 ÷ 30 = ¥0.0004), 不值当. 先用生命周期就行.

## 其它 OSS 前缀要不要也加生命周期?

OSS 里其它前缀的用途 + 建议:

| 前缀 | 用途 | 建议 |
|---|---|---|
| `avatars/` | 用户头像 | **不删** (用户长期资产) |
| `covers/` | 生成的封面图 | **不删** (用户长期资产) |
| `narration-output/` | 数字人/口播成片 | 30 天后删 (用户应已下载) |
| `voice-clones/` | 克隆音色样本 | **不删** (用户付费资产) |
| `whiteboard-bg/` | 白板背景图库 (admin 传) | **不删** |
| `desktop_release/` | .exe / latest.yml | **不删** |
| `recordings/` | 录屏临时文件 | **24h 删 ← 这次配的** |

按需逐个加规则即可.

5 分钟, 永久生效, 省 ¥120/月.
