## 9. 一堆 fix

**0a3e053 — api/proxy.ts Buffer 类型修**: Vercel build TypeScript 6.0.3 stricter,
Buffer 不在 BodyInit union 里. 两处 `body = buffer` 改 `body = new Uint8Array(buffer)`
(Buffer 本身 extends Uint8Array, 不复制内容).

**a2e0481 — main.py 没 import Request**: 用了 `request: Request` 注入但 import
里缺 Request → FastAPI 当成 Pydantic 模型找 body 数据 → 422 Unprocessable.
用户表现: 登录后账户中心打不开 + admin 显示"权限不足" (其实是 me API 挂了).
一行 import 修.

**6a06466 — main.py 加 __main__ 入口**: 之前只能 `uvicorn main:app --host 0.0.0.0
--port 18765` 启 (依赖一键启动.bat), 现在 `python main.py` 也能跑.

**25f6942 (后半) — index-server.py 加 HF_HUB_OFFLINE=1 + TRANSFORMERS_OFFLINE=1**:
用户报"index 一直在跑". 实际是启动时 transformers 库去 huggingface.co 验证
w2v-bert-2.0 preprocessor_config.json, 国内 SSL 不稳重试 5 次 (1+2+4+8+8 = 23s).
模型本地已下载, 离线模式直接用缓存, 启动快 20+ 秒.
