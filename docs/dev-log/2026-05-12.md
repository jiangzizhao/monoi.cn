### Stage 2.3: publish_xhs / publish_douyin (实际发布函数)
拿到 DOM dump 后写的实际 publish 函数, 档 B 策略 (停在'发布'按钮前, 不替用户点):

**小红书 selector**:
| 元素 | Selector | 备注 |
|---|---|---|
| 文件 input | `input.upload-input` | 隐藏 input, set_input_files 喂 |
| 标题 | `input[placeholder*="标题"]` | placeholder "填写标题会有更多赞哦" |

**抖音 selector**:
| 元素 | Selector | 备注 |
|---|---|---|
| 文件 input | `input[type="file"]` first | visible=True, 直接喂 |
| 标题 | `input[placeholder*="作品标题"]` | 抖音默认填了文件名, 我先 Ctrl+A+Delete 清空再 human_type |
| 描述 | `div.editor-comp-publish[contenteditable="true"]` | data-placeholder="添加作品简介" |

两个函数同结构, 流程:
1. set_input_files 喂视频
2. 等表单 (title input visible, 最多 5 分钟)
3. 拟人化填标题
4. 拟人化填描述+标签 (标签以 ` #tag1 #tag2 ` 附在描述末尾)
5. mouse_jitter 收尾
6. `wait_for_event('close', timeout=wait_close_timeout)` 等用户审稿点发布 / 关 Edge / 超时

默认 timeout 后来从 600s 改成 1800s (抖音 HD 视频转码可能 5-15 分钟, 用户审稿 + 等转码完成至少要 30 分钟).

## 2. 找素材按说话节奏拆 segment

用户反馈"找素材的断句不对, 应该按人说话那样断". 路径 A (合成完接素材) 直接用 whisper ASR 给的 segments, 但 whisper 默认按"长停顿 (1.5-2s+)"拆, 太粗.

加 `splitSegmentByPause(seg, gapThreshold, minDuration)` 工具:
- 遍历 seg.words, 看每个 word.end → next_word.start 的 gap
- gap ≥ 0.25s 且当前 buf 累计 ≥ 1s 时拆新 segment
- 短句不拆, 避免一镜素材太短

例: 6 秒原 segment 通常拆成 3-4 个 segment, 每个配不同 b-roll, 剪辑感更强.

阈值参数化, 后续不满意调 0.25 / 1.0 即可.

## 3. 口播剪辑气口/嗯嗯检测调激进

用户反馈"嗯嗯没去掉, 气口没去掉". 后端 3 个检测过保守, 改激进:

| 参数 | 现 | 改 | 影响 |
|---|---|---|---|
| silence_detect noise_db | -25 dB | -30 dB | 更轻的气口判 silence |
| silence_detect min_silence | 0.4s | 0.3s | 短停顿也算 |
| word_gaps min_gap | 0.4s | 0.3s | 词间短停顿也建议删 |
| _FILLER_WORDS | 嗯啊呃哦... | 加"恩"/"嗯哼"/"嗯啊"等 | whisper 中文常把"嗯"识别成同音"恩" |
| _detect_fillers 跳过 | 首+尾 | 只跳首 | 收尾"嗯..."没保护必要 |

3 处调用点 (clean_narration / clean_narration_video / clean_narration_video_oss) 全部同步.

## 5. 商业化体系 (重头戏)

### 5a. 设计 v1 落地 (docs/business-model.md)

跟用户多轮讨论确定:
- **4 档套餐**: Free (50 体验积分 + 水印) / Pro ¥99/月 (1500 分) / Max ¥199/月 (4000 分) / 旗舰年卡 ¥2980 (5000/月)
- **基础功能不扣积分**: 文案 / 素材匹配 / 封面 / 自动发布 — 当福利
- **高成本扣积分** (¥1=10 积分锚定): 配音 0.5-1.5/秒, 数字人合成 2/秒, 一键合成 10/次
- **4 档积分包**: ¥9.9-499, 月会员买积分有阶梯折扣 (Pro ¥1=15, Max ¥1=20, 旗舰 ¥1=25)
- **三级推广**:
  - 普通用户: 积分奖励 (注册 30 / 推荐买月卡 30% 等值积分 / 旗舰封顶 3000)
  - 认证推广员 (累计 5 付费用户 / ¥500): 现金 30% 首单 + 10% × 3 月续费
  - 核心合伙人 (月推 20 人 / ¥3000): 现金 50% 首单 + 15% × 3 月续费
- **续费分成只算 3 个月** (避免长期被薅光 LTV)
- **不退款** (一旦付费不可退)
- **支付**: 微信 + 支付宝双通道 (用户有商户号 + 跑通过)
