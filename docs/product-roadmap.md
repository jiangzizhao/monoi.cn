完整页 `/app/account/*` 6 个 tab:

2. **会员中心** `/membership` — 当前套餐 + 用量进度 + 4 档套餐对比 + 详细功能表 + FAQ + 续费/升级/取消自动续费
3. **充值积分** `/credits` — 4 档积分包 + 扣减规则说明 + 折扣阶梯
4. **消费记录** `/transactions` — Tab: 积分流水 / 订单记录 / 导出 CSV
5. **我的推广** `/referral` — 推广码 + 海报生成 + 等级 + 累计 + 收益 + 提现 + 推荐列表
6. **安全设置** `/security` — 改密码 / 登录设备 / 双因素 / 微信绑定 / 支付宝绑定

### B. 登录系统增强

新增 3 种登录方式:
1. **手机号+短信验证码** (中国用户习惯)
2. **微信扫码登录** (Web 端 OAuth)
3. **微信内置浏览器一键登录** (公众号场景, V2)

辅助:
4. **忘记密码** (邮件链接 / 手机短信二选一)
5. 注册流程加手机号绑定 (可选, 不强制)

### C. 管理员后台 `/admin`

只允许 `is_admin = 1` 的用户访问, 走主应用同 domain (`monoi-cn.vercel.app/admin`), JWT 鉴权.

5 大块:

1. **数据看板** `/admin/dashboard`
   - 累计用户数 / 付费转化率 / DAU / MAU
   - 营收日报/月报 (按套餐/积分包/支付方式分组, recharts 折线图 + 柱状图)
   - 留存曲线 (D1/D7/D30)
   - 推广员转化漏斗 (普通 → 认证 → 合伙人)

2. **用户管理** `/admin/users`
   - 用户列表 (搜索: 用户名/邮箱/手机, 筛选: 套餐/付费/注册时间/推广来源)
   - 用户详情 (注册信息/手机/邮箱/绑定/订单历史/积分流水/推广关系)
   - **手工操作**:
     - 升级/降级套餐 (含设置到期时间)
     - 加减积分
     - 升认证推广员
     - 升核心合伙人
     - 封禁账号
     - 重置密码 (发邮件/短信)

3. **订单管理** `/admin/orders`
   - 全部订单 (筛选: 状态/金额/支付方式/套餐)
   - **特殊退款处理** (政策不退, 但极端情况人工)
   - 异常订单 (支付了未到账等)

4. **推广员管理** `/admin/referrers`
   - 推广员列表 (按等级)
   - 提现申请 (待审 / 已审 / 已打款)
   - 标记打款状态 + 上传打款凭证截图
   - 异常推广检测 (单 IP 多注册 / 退费率高)

5. **系统配置** `/admin/config`
   - 套餐价格 / 积分扣减规则 / 推广分成比例 (热改, 改了立刻生效)
   - 公告栏 (用户登录后弹窗)
   - 黑名单关键词 (用户发布描述出现就拦截)

### D. 第三方集成

1. **微信支付 + 支付宝** (用户有商户号 + 跑通过)
2. **阿里云短信服务** (注册/换绑/忘记密码)
3. **微信开放平台 OAuth** (Web 扫码登录)
4. **微信公众号** (内嵌跳转登录, V2)

### 商业化原则 (前面 docs/business-model.md 已定)
- 4 档套餐 (Free / Pro ¥99/月 / Max ¥199/月 / 旗舰 ¥2980/年)
- 4 档积分包 (¥9.9 / ¥49 / ¥199 / ¥499)
- 积分锚定: ¥1=10 积分 (会员买积分有阶梯折扣)
- 续费分成只算 3 个月
- 不退款 (一旦付费不可退)
- 普通用户推广拿积分 (零合规), 认证推广员+ 拿现金 (≥¥100 提现)
- 三级推广员等级 + 自动升级判定

## 待补充的产品细节 (Phase 1 开干前要定)

### 1. 套餐"按月/按年" toggle 要不要?
- 现状: Pro/Max 只有月付, 旗舰只有年付
- 加年付选项 (Pro ¥990/年 省 17% / Max ¥1990/年 省 17%)? 可增加锁年率, 但 schema 要加 pro_yearly / max_yearly 两档

### 2. 旗舰套餐增强权益
当前旗舰 ¥2980/年 比 Max 仅多: 数字人 100→300/月 + 克隆 3→5 + 团队多席位. 建议加:
- VIP 专属客服 (微信 1v1)
- 提前体验新功能 (新数字人 / 新 TTS 模型)
- API 访问 (V2 优先开)
- 不限制单视频时长 (其他套餐限 5 分钟)
- 推荐分成提升 (作为旗舰用户帮推, 一次性 30%)

### 3. 注册流程要不要强制手机号?
- 选项: 邮箱 only / 邮箱+手机可选 / 强制手机
- 中国用户习惯: 强制手机号 + 验证码注册

### 4. 自动续费默认开启还是关闭?
- 开启 (流失低, 用户可能忘记取消, 投诉风险)
- 关闭 (用户主动续费, 流失高但合规)
- 折中: 默认开启, 显著入口可关 (页面 + 邮件提醒)

### 5. 退款政策细节
- 不退款 = 完全不退? 还是"未消费积分 7 天内可退"? (常见做法)

### 6. 海报生成 (推广用)
- 自动生成带二维码的海报图给推广员, 直接发朋友圈
- 海报模板要不要做? 工时 +4h
